# 🎓 Student Attendance Management System

A web-based Student Attendance Management System developed using PHP, MySQL, HTML, CSS, Bootstrap, and JavaScript.

## ✨ Features

- 🔐 Admin Login
- 📊 Professional Dashboard
- ➕ Add Student
- 👨‍🎓 View Students
- ✏️ Edit Student Details
- 🗑️ Delete Student
- 📝 Mark Attendance
- 📈 Attendance Report
- 📱 Responsive User Interface

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- XAMPP
